
# Code of Conduct

**"It’s not about you, it’s about the work."**

This project is a meritocracy. Contributions are judged solely on technical merit — the best solution wins.

### Core Principles

- Focus on technical matters: code, architecture, performance, correctness, and maintainability.
- The best solution wins. Objective quality decides acceptance.
- If your idea is rejected or improved, treat it as a learning opportunity. Do not take it personally.
- If your solution is accepted, do not gloat. Use it to help improve the project and others.
- Expect rigorous feedback on your work. Use it to produce better contributions.
- Submit clear, tested, well-documented changes that add real value.

Bring your best every time.
